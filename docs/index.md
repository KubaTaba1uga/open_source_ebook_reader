Open Source eBook Reader
=================================

This is open source ebook reader project. 
Goal of the project is to allow users to hack their readers easily.

I have a kobo reader which is pretty cool, but lack some features which are realtively easy to implement on linux, like words translations between languages in pdfs, or auto syncing books on the device with git. The idea is to create a platform which integrates Linux supported CPU, epaper display, buttons, battery and an enclosure, and on top of the platform build ebook reader app, which can be easily customized.

If you just need an ebook reader, going with Kobo is better choice for 99% of use cases, but if you are a programmer with some wierd ideas how would you like to use the reader, feel free to continue. Building the device itself is pretty time consuming, explaining it all and documenting would require a lot of efforts wich i would rather put into improving the software, so at the moment only channel to aquire the device is to buy it. If you are interested in buying the reader just email me directly, and i will build one for you.
